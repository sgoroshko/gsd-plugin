## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-11-01 | Tampering | path inputs to detect()/load() | mitigate | Reuse `conventions.sanitizePaths` + `SAFE_PATH_RE`; `path.resolve(cwd, ...)` confined; reject `..`/absolute before read |
| T-11-02 | Denial of Service | export/import/CRUD/TODO regexes on adversarial source | mitigate | Linear regexes (no nested quantifiers); single-pass imported-names Set (Pitfall 3, not O(files^2)); `MAX_SCAN_BYTES` cap |

| T-11-04 | Tampering (data) | TODO / CRUD strings inside source string literals | mitigate | Run `conventions.blankSpans` before every body/export/import scan |
| T-11-SC | Tampering | npm/pip/cargo installs | mitigate | Zero packages installed — Node built-ins + ./conventions.cjs only (RESEARCH Package Legitimacy Audit; no install task) |
